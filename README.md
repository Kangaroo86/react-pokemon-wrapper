#Pokemon Arena App
This is a Pokemon CRUD app that revolves around users building their own Pokemon deck to battle other players in real time!!

![screenshot](./src/images/giphy.gif?raw=true)

## Frontend Technology:
-React Redux,
-Semantic CSS,
-Socket.io,
-Webpack

## Backend Technology:
-PostgreSQL,
-Express,
-Node.js
-Knex
